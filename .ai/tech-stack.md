# Technology Stack

## Frontend
**Astro 5** + **React 19** - fast pages with interactive components  
**TypeScript 5** - typing and better IDE support  
**Tailwind 4** - utility-first CSS  
**Shadcn/ui** - ready-made UI components

## Backend
**Supabase** - PostgreSQL + BaaS + Auth (open source, self-hosting capability)

## Testing
**Vitest** - unit and integration testing framework  
**React Testing Library** - testing React components  
**Playwright** - end-to-end testing framework

## Deployment
**GitHub Actions** - CI/CD  
**DigitalOcean** - hosting (Docker)
